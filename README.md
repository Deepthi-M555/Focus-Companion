# 🚀 Focus Companion

A full-stack desktop productivity application designed to help users stay focused, manage tasks, and improve productivity through structured scheduling and focus sessions.

---

## 🧠 Features

- 🔐 JWT-based authentication system
- 📝 Task creation, update, and deletion
- 📊 User-specific task management (secure & isolated)
- ⏱️ Focus session tracking (in progress)
- 🖥️ Desktop integration using Electron (planned)

---

## 🏗️ Tech Stack

**Frontend:**
- React (Vite)

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB (Mongoose)

**Authentication:**
- JWT
- bcrypt
